# - User Guide -
ðŸ“Ž Type regular commands with flags such as ls, and chain them with | between them. <br>
```
ls -l
ls | grep myfile
```
ðŸ“Ž Change prompt <br>
```
prompt = good morning:
```
ðŸ“Ž Redirect io by typing after the last command <, >>, >, or 2> <br>
```
date > todayis
cat < todayis
ls badfilename 2> errormsg
```
ðŸ“Ž Perform certain commands with an IF block, the format is as follows <br>
- if (cmd)
- then
- (cmds or empty)
- else
- (cmds or empty)
- fi
```
if ls | grep badfile
then
rm badfile
echo file removed
else
echo no such file
fi
```
```
if date | grep Sat
then
echo Shabat Shalom
else
fi
```
ðŸ“Ž Initialize local variables, single or multiple, $name = val, or by read v1 v2 and after enter typing the values <br>
```
$cmd = ls
$cmd (performs ls and usual)
read cmd flags
$cmd $flags (performs ls with the flag -l)
```
ðŸ“Ž Navigate previous commands by (UP, then ENTER) or (DOWN, then ENTER) append and execute again <br>
```
ls
UP arrow key, enter
ls -l
```
